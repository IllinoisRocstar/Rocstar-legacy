Integration Interface

The Integration Interface presentation goes through the software integration substrate and integration through Roccom. Data attributes, inheritance, function and runtime are all discussed here.

Rocman Orchestration

The Rocman Orchestration presentation goes through the component management and orchestration framework of Rocman. The hierarchical architecture as well as an example are discussed.

Service Components

The Service Components presentation discusses the surface propagation and data transfer components of the Rocstar suite. Various aspects from the utilities and I/O services are mentioned while interface data transfer and mesh enhancement and modification are also discussed in greater detail.

Rocman Paper

The Rocman Paper gives information into the upcoming generation of Rocman built on top of the Roccom integration framework.

Supported Roccom File Formats

The FileFormats developer guide gives insight into the conventions used by roccom-compliant data files and modules such as Rocin and Rocout. The guide delves into supported formats such as HDF4 and CGNS.

Rocface Surfdiver Module

Surfdiver is an executable programs built on top of Rocface for constructing a common refinement or common tessellation of two nonmatching surface meshes. The documents describes the file formats, procedure for building and running, and advanced procedures for tuning feature-detection parameters for Surfdiver.

Rocstar Time Zooming and Rocket Burnout

Time Zooming gives a user and developer report for time zooming in Rocstar. The focus, here, in on testing, using and implementing the time zooming aspect of the Rocstar simulation suite. Tests, trials and behaviors are summarized accordingly as well.
